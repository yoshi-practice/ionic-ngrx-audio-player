# ionic-ngrx-audio-player
This is an audio player application that developed by ionic, NgRx, RxJs, Angular.

---
## How to Run

### Local

```
# iOS
$ ionic cordova run ios

# android
$ ionic cordova run android
```

### Docker
```
$ docker build -t audio-player .
$ docker run -it audio-player
# ... Local command
```
